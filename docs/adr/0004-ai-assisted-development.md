# ADR 0004: AI-Assisted Development Assets

## Status

Accepted

## Decision

Codify repository-specific AI guidance through Copilot instructions, path-specific instructions, prompts, agents, hooks, skills, and documentation.

## Consequences

- AI tools receive concrete constraints instead of generic coding advice.
- Repeated feature work follows consistent Clean Architecture patterns.
- Humans remain responsible for review, tests, security, and production readiness.
